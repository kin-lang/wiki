# Intro

`KIN_IGIHE()` in-built utility provides access to date and time methods that can help you deal with the time.

## How to

- `KIN_IGIHE.isaha()` : Returns current time in format `(HH:MM:SS)`
- `KIN_IGIHE.umunsi()` : Returns the current day, ex: `Monday, Tuesday ...`
- `KIN_IGIHE.itariki()` : Returns the current date in format `(YYYY-MM-DD)`
