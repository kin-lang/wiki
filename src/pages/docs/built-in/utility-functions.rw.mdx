# Intangiriro

Dore urutonde rw’izindi **method** zubatsemo tutarigeze tuvuga muri iyi documentation.

# Method

## `ubwoko()`

Iyi **method** ikoreshwa kugira ngo imenye **ubwoko** bw’icyo watanze.

Urugero:

```Kin
reka x = 5
tangaza_amakuru(ubwoko(x))
```

> Ibi bizasohora `umubare`.

## `sisitemu()`

Iyi **method** ikoreshwa mu gukora amategeko ya sisiteme ava muri porogaramu ya Kin.

Urugero:

```Kin
sisitemu("sudo shutdown now")
```

> Uyu murongo uzafunga mudasobwa.

## `hagarara()`

Iyi **method** ikoreshwa guhagarika Kin igihe iri gukora.

Urugero:

```Kin
hagarara(0)
```

> Ibi bizahagarika porogaramu ya Kin, kandi byerekana ko ishyirwa mu bikorwa ryagenze neza.

Iyi **function** yubatswemo yakira **argument**, ari yo nimero:
`0` ku ishyirwa mu bikorwa ryagenze neza
`1` ku ishyirwa mu bikorwa ritagenze neza.
