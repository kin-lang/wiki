Muri ururimi rwa porogaramu, aya mategeko yitwa **_statements_**.  
Porogaramu ya **_Kin_** ni urutonde rwa **statements** za porogaramu.

## Semicolons ;

Muri byinshi mu ndimi za porogaramu nka C, C++, ... semicolon ikoreshwa kurangiza statement.  
Muri **_Kin_**, semicolon si ngombwa. Ariko hari aho uyikeneye.

- Semicolon ikenewe igihe utanze variable ariko utayihaye agaciro.

```Kin
reka x; # Bizakora
reka x  # Ntibizakora
```

- Semicolon ikenewe igihe function isubiza ariko nta expression isubiza.

```Kin
porogaramu_ntoya main() {
    tanga; # Bizakora
}
```

Muri rusange, semicolon ikoreshwa kugira ngo ibwire **_Kin_** ko hari statement yacitse.

## Intera (White Spaces)

**_Kin_** ireka intera nyinshi idakurikira.
Ushobora kongeramo intera muri script yawe kugirango isomeke neza.
Imirongo ikurikira irasa:

```Kin
reka x = 5
reka x=5
```
