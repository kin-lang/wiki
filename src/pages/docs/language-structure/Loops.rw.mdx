# Intangiriro

**Loops** zikoreshwa mu gusubiramo igice cya **code** igihe **condition** runaka iba ari ukuri.

## Imiterere (Syntax)

```Kin
subiramo_niba (condition) {
  # igice cya code kizakorwa
}
```

## Urugero

```Kin
reka x = 0
subiramo_niba (x < 100) {
  x = x + 1
  tangaza_amakuru(x)
}
```

> Iyi code izacapura 1, 2, 3, ..., 98, 99, 100

**Condition** ishobora kuba **complex** uko ushaka!
