# Intangiriro

**Insinga (Strings)** zikoreshwa mu kubika inyandiko (text).

Insinga zandikwa ukoresheje **double quotes**.

Urugero:

```Kin
reka x = "Hello World"
```

## Uburyo bwo Gukora ku Insinga (String Methods)

**_Kin_** itanga uburyo butandukanye bwo gucunga insinga, byose biri muri **`KIN_AMAGAMBO`** utility y’imbere muri **_Kin_**.

### Guhuza Insinga (String Concatenation)

Ushobora guhuza insinga ukoresheje **`KIN_AMAGAMBO.huza()`**.

Urugero:

```Kin
reka x = "Hello "
reka y = "World"

reka z = KIN_AMAGAMBO.huza(x, y)
tangaza_amakuru(z)
```

> Output: Hello World

> Inyandiko: **`KIN_AMAGAMBO.huza()`** ishobora kwakira arguments nyinshi uko ushaka.

### Uburebure bw'Insinga (String Length)

Ushobora kumenya uburebure bw’insinga ukoresheje **`KIN_AMAGAMBO.ingano()`**.

Urugero:

```Kin
reka ijambo = "Hello World"
tangaza_amakuru(KIN_AMAGAMBO.ingano(ijambo))
```

> Output: 11

### Gutandukanya Insinga (Splitting String)

Ushobora gutandukanya insinga ukoresheje delimiter icyo ari cyo cyose ukoresheje **`KIN_AMAGAMBO.tandukanya()`**.

Urugero:

```Kin
reka ijambo = "Hello World"
reka hello = KIN_AMAGAMBO.tandukanya(ijambo, " ")
tangaza_amakuru(hello[0])
```

> Output: Hello

> _Inyandiko_: **`KIN_AMAGAMBO.tandukanya()`** isubiza **array**, kandi iyo ugerageje kuyicapura yose, bizatanga `[Object Object]`. Ibi biterwa n’uko **Arrays** ari **Objects** muri Kin. Ugomba gukorana n’igice kimwe cya array icyarimwe.

### Inyuguti Ntoya n’Inyuguti Nkuru (Upper Case & Lower Case)

- Inyuguti Nkuru (UpperCase)

```Kin
KIN_AMAGAMBO.inyuguti_nkuru(string) # Iyi izahindura insinga yahawe inyuguti nkuru
```

- Inyuguti Ntoya (LowerCase)

```Kin
KIN_AMAGAMBO.inyuguti_ntoya(string) # Iyi izahindura insinga yahawe inyuguti ntoya
```

### Gufata Inyuguti imwe (Getting Character)

Ushobora gufata inyuguti iri ku index runaka ukoresheje **`KIN_AMAGAMBO.inyuguti(string, characterIndex)`**.

Urugero:

```Kin
reka str = "this is a string"
reka char = KIN_AMAGAMBO.inyuguti(str, 8) # Izagarura 'a'
tangaza_amakuru(char)
```

> Ibi bizacapura inyuguti `a` kuri console.

## Gusubiramo Inyuguti z’Insinga (Iterating over String)

Muri **_Kin_**, ukoresheje **`KIN_AMAGAMBO.ingano()`** na **`KIN_AMAGAMBO.inyuguti()`**, birashoboka gusubiramo inyuguti z’insinga.

Urugero:

```Kin
reka str = "a string"

reka i = 0

subiramo_niba(i < KIN_AMAGAMBO.ingano(str)) { # ukoresha KIN_AMAGAMBO.ingano() kumenya uburebure bw’insinga
  tangaza_amakuru(KIN_AMAGAMBO.inyuguti(str, i)) # Fata inyuguti iri ku index runaka
  i = i + 1
}
```

> Output: buri nyuguti icapurwa ku murongo wayo.
