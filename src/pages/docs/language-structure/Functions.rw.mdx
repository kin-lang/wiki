# Intangiriro

**_Kin_** function ni igice cya code cyateguwe gukora umurimo runaka.

**_Kin_** function ikora iyo "ikintu" kiyihamagaye (yahamagawe).

Urugero:

```Kin
porogaramu_ntoya function(a, b) {
  tanga a * b
}
```

## Imiterere ya Function (Function Syntax)

**_Kin_** function ishyirwaho hifashishijwe ijambo `porogaramu_ntoya`, rikurikirwa n’izina, hanyuma hakurikiraho parentheses ().

Amazina ya function ashobora kugira inyuguti, imibare, underscores, na dollar signs (amategeko ameze nk’ayo ku variables).

Mu parentheses hashobora gushyirwa amazina y’parameters atandukanijwe na comma:
(parameter1, parameter2, ...)

Code izakorwa na function ishyirwa muri curly brackets: {}

```Kin
porogaramu_ntoya izina(parameter1, parameter2, parameter3) {
  # code izakorwa
}
```

Parameters za function zishyirwa muri parentheses () mu gusobanura function.

Arguments za function ni agaciro function yakiriye igihe yahamagajwe.

Muri function, arguments (parameters) bikora nka variables z’imbere muri function (local variables).

## Guhamagara Function (Function Invocation)

Code iri muri function izakora igihe "ikintu" cyahamagaje (calls) function:

```Kin
izinaFunction(argument1, argument2, argument3, ...)
```

## Gusubiza Agaciro (Function Return)

Iyo **_Kin_** igeze ku nteruro ya `return`, function irahagarara gukora.

Niba function yahamagajwe mu nteruro, **_Kin_** izasubira ku nteruro ikurikira iyihamagaje.

Functions akenshi zibarira agaciro kagarurwa. Agaciro karasubizwa ku "wahamagaye":

```Kin
porogaramu_ntoya myFunction(a, b) {
  # Function isubiza igiteranyo cya a na b
  return a * b;
}

# Function ihamagajwe, agaciro gasubijwe kazajya muri x
reka x = myFunction(4, 3);
```

> Nyuma ya `return` ntihagire indi nteruro iri munsi yayo muri function

Ibi rero bizatanga ikosa:

```Kin
porogaramu_ntoya function(a, b) {
  tanga a*b
  tangaza_amakuru("Igiteranyo ni ", a*b)
}
```

## Impamvu yo Gukoresha Functions

Ukoresheje functions ushobora gusubiramo code.

Ushobora kwandika code ishobora gukoreshwa inshuro nyinshi.

Ushobora gukoresha code imwe n’arguments zitandukanye, kugira ngo uhabwe ibisubizo bitandukanye.
