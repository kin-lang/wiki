# Intangiriro

`KIN_IGIHE()` yubatsemo igufasha kubona uburyo bwo gukoresha itariki n’igihe, bigufasha gukora ibikorwa bijyanye n’igihe.

## Uko bikorwa

- `KIN_IGIHE.isaha()` : Isubiza isaha iri muri format `(HH:MM:SS)`
- `KIN_IGIHE.umunsi()` : Isubiza umunsi turiho, urugero: `Kuwa Mbere, Kuwa Kabiri ...`
- `KIN_IGIHE.itariki()` : Isubiza itariki turiho muri format `(YYYY-MM-DD)`
