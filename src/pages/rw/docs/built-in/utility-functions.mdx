# Intangiriro

Uru ni urutonde rw’izindi **method** zubatsemo tutigeze tuvugaho muri iyi documentation.

# Method

## `ubwoko()`

Iyi **method** ikoreshwa kugira ngo imenye **ubwoko** bw’icyo watanze.

Urugero:

```Kin
reka x = 5
tangaza_amakuru(ubwoko(x))
```

> Ibi bizasohora `umubare`.

## `sisitemu()`

Iyi **method** ikoreshwa mu gukora amategeko ya sisiteme ava muri porogaramu ya Kin.

Urugero:

```Kin
sisitemu("sudo shutdown now")
```

> Uyu murongo uzazimya mudasobwa.

## `hagarara()`

Iyi **method** ikoreshwa guhagarika Kin igihe iri gukora.

Urugero:

```Kin
hagarara(0)
```

> Ibi bizahagarika porogaramu ya Kin, kandi byerekana byagenze neza.

Iyi **function** yubatswemo yakira **igipimo (argument)**, ari yo nimero:
`0` niba ushaka ko bigenda neza
`1` niba ushaka ko bitagenda neza.
