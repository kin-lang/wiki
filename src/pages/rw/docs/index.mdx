# Murakaza neza ku bisobanuro bya Kin

Kin ni ururimi rw'iporogaramu rwagenewe gufasha Abanyarwanda kwiga porogaramu mu rurimi rwabo. Rurashingiye ku bworoherane no kwigisha.

## Ibintu by'ingenzi

- **Bworoshye**: Kin ifite imvugo yoroshye kandi yumvikana
- **Mu Kinyarwanda**: Amagambo yose y'iporogaramu ari mu Kinyarwanda
- **Kwigisha**: Igenewe cyane cyane kwigisha no kwiga iporogaramu
- **Gushya**: Ikoresha tekinoroji zigezweho

## Tangira

Kugira ngo utangire gukoresha Kin, soma [amabwiriza yo gutangira](/rw/docs/getting-started).

## Imiterere y'ururimi

Menya [imiterere y'ururimi rwa Kin](/rw/docs/language-structure) kugira ngo usobanukirwe neza uko rukora.

## Ibikorwa byubatswe

Kin ifite [ibikorwa byinshi byubatswe](/rw/docs/built-in) bigufasha gukora porogaramu zitandukanye.
