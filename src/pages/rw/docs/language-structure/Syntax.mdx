Muri ururimi rwa porogaramu, aya mategeko yitwa **_amatangazo_**. Porogaramu ya **_Kin_** ni urutonde rw' **amatangazo** za porogaramu.

## Akadomo n'akitso (Semicolons) ;

Mu ndimi nyinshi za porogaramu nka C, C++, ... akadomo n'akitso bikoreshwa kurangiza itangazo(statement).  
Muri **_Kin_**, akadomo n'akitso si ngombwa. Ariko hari aho bikenewe.

- Akadomo n'akitso bikenewe igihe utanze ihinduragaciro(variable) ariko ntuyihe agaciro.

```Kin
reka x; # Bizakora
reka x  # Ntibizakora
```

- Akadomo n'akitso bikenewe mugihe umumaro(function) usubije ariko nta mvugogaciro(expression) ihari yo gutanga.

```Kin
porogaramu_ntoya main() {
    tanga; # Bizakora
}
```

Muri rusange, akadomo n'akitso bikoreshwa kugira ngo bibwire **_Kin_** ko hari itangazo ryasimbutswe.

## Intera (White Spaces)

**_Kin_** yirengangiza intera nyinshi zikurikirana. Ushobora kongeramo intera muri porogaramu yawe kugirango isomeke neza.
Imirongo ikurikira irasa:

```Kin
reka x = 5
reka x=5
```
