# Intangiriro

**Ijambo/Interuro (Strings)** zikoreshwa mu kubika inyandiko (text).

Ijambo/Interuro zandikwa ukoresheje **utwuguruzo n'utwugarizo (double quotes)**.

Urugero:

```Kin
reka x = "Hello World"
```

## Uburyo bwo Gukora ku Ijambo/Interuro (String Methods)

**_Kin_** itanga uburyo butandukanye bwo gucunga Ijambo/Interuro, byose biri mu mumaro wubatswe muri **_Kin_** ari wo **`KIN_AMAGAMBO`**.

### Guhuza Ijambo/Interuro (String Concatenation)

Ushobora guhuza Ijambo/Interuro ukoresheje **`KIN_AMAGAMBO.huza()`**.

Urugero:

```Kin
reka x = "Hello "
reka y = "World"

reka z = KIN_AMAGAMBO.huza(x, y)
tangaza_amakuru(z)
```

> Ibisohoka (Output): Hello World

> Icyitonderwa: **`KIN_AMAGAMBO.huza()`** ishobora kwakira ibipimo(arguments) byinshi uko ubishaka.

### Uburebure bw'Ijambo/Interuro (String Length)

Ushobora kumenya uburebure bw’ijambo/interuro ukoresheje **`KIN_AMAGAMBO.ingano()`**.

Urugero:

```Kin
reka ijambo = "Hello World"
tangaza_amakuru(KIN_AMAGAMBO.ingano(ijambo))
```

> Ibisohoka: 11

### Gutandukanya Ijambo/Interuro (Splitting String)

Ushobora gutandukanya ijambo/interuro ukoresheje Ikimenyetso gitandukanya (delimiter) icyo ari cyo cyose **`KIN_AMAGAMBO.tandukanya()`**.

Urugero:

```Kin
reka ijambo = "Hello World"
reka hello = KIN_AMAGAMBO.tandukanya(ijambo, " ")
tangaza_amakuru(hello[0])
```

> Ibisohoka (Output): Hello

> _Icyitonderwa_: **`KIN_AMAGAMBO.tandukanya()`** isubiza **urutonde rw'ibintu (array)**, kandi iyo ugerageje kubyandika ku igaragazamakuru, bizatanga `[Object Object]`. Ibi biterwa n’uko **Arrays** ari **Objects** muri Kin. Ugomba gukorana n’igice kimwe cy'urutonde rw'ibintu icyarimwe.

### Inyuguti Ntoya n’Inyuguti Nkuru (Upper Case & Lower Case)

- Inyuguti Nkuru (UpperCase)

```Kin
KIN_AMAGAMBO.inyuguti_nkuru(string) # Iyi izahindura Ijambo/Interuro yahawe mo inyuguti nkuru
```

- Inyuguti Ntoya (LowerCase)

```Kin
KIN_AMAGAMBO.inyuguti_ntoya(string) # Iyi izahindura Ijambo/Interuro yahawe mo inyuguti ntoya
```

### Gufata Inyuguti imwe (Getting Character)

Ushobora gufata inyuguti iri ku umwanya runaka ukoresheje **`KIN_AMAGAMBO.inyuguti(string, characterIndex)`**.

Urugero:

```Kin
reka str = "this is a string"
reka char = KIN_AMAGAMBO.inyuguti(str, 8) # Izagarura 'a'
tangaza_amakuru(char)
```

> Ibi bizandika inyuguti `a` kuri console.

## Gusubiramo Inyuguti y’Ijambo/Interuro (Iterating over String)

Muri **_Kin_**, ukoresheje **`KIN_AMAGAMBO.ingano()`** na **`KIN_AMAGAMBO.inyuguti()`**, birashoboka gusubiramo inyuguti y’ijambo/interuro.

Urugero:

```Kin
reka str = "a string"

reka i = 0

subiramo_niba(i < KIN_AMAGAMBO.ingano(str)) { # ukoresha KIN_AMAGAMBO.ingano() kumenya uburebure bw’ijambo/interuro
  tangaza_amakuru(KIN_AMAGAMBO.inyuguti(str, i)) # Fata inyuguti iri ku umwanya runaka
  i = i + 1
}
```

> Ibisohoka: buri nyuguti irandikwa ku murongo wayo.
