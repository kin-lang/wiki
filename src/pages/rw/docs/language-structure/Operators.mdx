title: Ibimenyetso Bikora (Operators)

# Intangiriro

**_Kin_** ifite urutonde rw’ibimenyetso Bikora bikora ibintu byinshi bitandukanye.

Muri iki gitabo, turasobanura uburyo **_Kin_** ikoresha Ibimenyetso Bikora mu gukora ibikorwa bitandukanye.

## Ibimenyetso by’Imibare (Arithmetic Operators)

Ibimenyetso by’imibare bikora ibikorwa by’imibare ku mibare.

Urugero:

```Kin
x = 1 + 5
````

> Ikimenyetso + cyakoreshejwe mu kongeraho 1 kuri 5

| Ikimenyetso | Ibisobanuro                                |
| ----------- | ------------------------------------------ |
| +           | Guteranya                                  |
| -           | Gukura                                     |
| \*          | Gukuba                                     |
| /           | Kugabanya                                  |
| ^           | Gukuba inshuro nyinshi (kubara ubushobozi) |
| %           | Ibisigazwa byo kugabanya (modulus)         |

> **Icyitonderwa**: Nugerageza gukoresha ibimenyetso by’imibare ku mirongo y’inyuguti (strings) uzabona `ubusa`. Ibikorwa by’imibare bikora gusa ku mibare n’imvugo zibarika (expressions).

## Ikimenyetso cyo Guhereza Agaciro (Assignment Operator)

Ikimenyetso cyo guha agaciro gikoresha mu guha ihinduragaciro (variable) agaciro.
***Kin*** ikoresha `=` nk’ikimenyetso cyo guha agaciro.

## Ibimenyetso byo Guhangana (Comparison Operators)

Ibimenyetso byo guhangana bikora mu kugereranya imvugo nyinshi.

| Ikimenyetso | Ibisobanuro                |
| ----------- | -------------------------- |
| ==          | bingana                    |
| !=          | bitangana                  |
| >           | biruta                     |
| \<          | biri munsi                 |
| >=          | biruta cyangwa bingana     |
| \<=         | biri munsi cyangwa bingana |

## Ibimenyetso by’Ubwenge (Logical Operators)

Ibimenyetso by’ubwenge bikora mu guhuza imvugo zishingiye ku byemezo (conditions).

| Ikimenyetso | Ibisobanuro |
| ----------- | ----------- |
| &&          | kandi       |
| \|\|        | cyangwa     |
| !           | oya (not)   |
