
# Intangiriro

**Urutonde rw'ibintu (array)** ni **ihinduragaciro** ridasanzwe kuko rushobora gufata agaciro kenshi:

Urugero:

```Kin
reka names  = ["Bwenge", "Nyabihu"]
```

## Kuki Dukoresha Urutonde rw'ibintu?

Niba ufite urutonde rw’ibintu (nk’urugero amazina y’imodoka), kubibika muri **minduragaciro (Variables)** zitandukanye byashoboka gutya:

```Kin
reka car1 = "Saab"
reka car2 = "Volvo"
reka car3 = "BMW"
```

Ariko se, niba ushaka guhitamo imodoka imwe runaka? Kandi ibaze noneho udafite imodoka 3 gusa, ahubwo ufite 300?

Igisubizo ni **Urutonde rw'ibintu**!

**Urutonde rw'ibintu** rushobora gufata agaciro kenshi munsi y’izina rimwe, kandi ushobora kubona agaciro ukoresheje **umwanya (index number)**.

## Gukora Urutonde rw'ibintu

Gukoresha **array literal** ni bwo buryo bworoshye bwo gukora urutonde rw'ibintu muri **_Kin_**.

**Syntax:**

```Kin
ntahinduka array_name = [item1, item2, ...]
```

Intera cyangwa umurongo ntibitera ikibazo. Ihamagarwa(Declaration) rishobora kugera ku mirongo myinshi:

```Kin
ntahinduka cars = [
  "Saab",
  "Volvo",
  "BMW"
];
```

Ushobora kandi gukora urutonde rw'ibintu hanyuma ukongeramo ibice:

```Kin
ntahinduka cars = [];
cars[0]= "Saab";
cars[1]= "Volvo";
cars[2]= "BMW";
```

## Kwinjira mu Bice by’Imbonerahamwe

Ushobora kubona agaciro k'urutonde rw'ibintu ukoresheje **umwanya(index number)**:

```Kin
ntahinduka cars = ["Saab", "Volvo", "BMW"];
reka car = cars[0];
```

> Umwanya w'urutonde rw'ibintu utangirira kuri 0.
> \[0] ni igice cya mbere. \[1] ni icya kabiri.

## Guhindura Agaciro k’Igice cy’Urutonde rw'ibintu

Iri tangazo rihindura agaciro k’igice cya mbere muri **cars**:

```Kin
reka cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Opel";
```

# Methode z’Urutonde rw'ibintu

### Ingano y'Urutonde rw'ibintu

Ushobora kubona **ingano** y'Urutonde rw'ibintu ukoresheje `KIN_URUTONDE.ingano()`.

Urugero:

```Kin
reka ijambo = ["Hello", "World"]
tangaza_amakuru(KIN_URUTONDE.ingano(ijambo))
```

> Ibisohoka: 2

### Kureba niba Urutonde rw'ibintu rurimo Agaciro

Ushobora kugenzura niba Urutonde rw'ibintu rurimo agaciro runaka ukoresheje `KIN_URUTONDE.ifite()`.

Urugero:

```Kin
reka ijambo = ["Hello", "World"]
tangaza_amakuru(KIN_URUTONDE.ifite(ijambo, "Hello"))
```

> Ibisohoka: nibyo

### Guhuza Ibice by'Urutonde rw'ibintu

Ushobora guhuza ibice byose by'Urutonde rw'ibintu kugira ngo bikore ijambo rimwe ukoresheje `KIN_URUTONDE.kora_ijambo()`.

Urugero:

```Kin
reka cars = ["Saab", "Volvo", "BMW"];
tangaza_amakuru(KIN_URUTONDE.kora_ijambo(cars))
```

> Ibisohoka(Output): SaabVolvoBMW
