# Intangiriro

**Iminduragaciro** ni ibikoresho byo **kubika amakuru**.

Muri **_Kin_**, iminduragaciro zishobora gutangazwa mu buryo bubiri:

## `reka`

Iyo utanze ihinduragaciro ukoresheje ijambo ry’ingenzi `reka`, ushobora guhindura agaciro ka ryo nyuma muri porogaramu.

```Kin
reka x = value
```

Ihinduragaciro `x` iratangazwa kandi yahawe n'agaciro ka `value`, ariko ako agaciro gashobora guhindurwa nyuma muri porogaramu.

Nanone, ushobora gutanga ihinduragaciro ntuyihe agaciro:

```Kin
reka x;
```

> Muri iki gihe, ihinduragaciro `x` iratangazwa ariko nta gaciro yahawe.
> **Icyitonderwa:** Ugomba kongeraho akadomo n'akitso **_(;)_** ku iherezo ry’iyi nteruro(statement).

## `ntahinduka`

Iyo utanze ihinduragaciro ukoresheje ijambo ry’ingenzi `ntahinduka`, ntabwo ushobora guhindura agaciro kayo nyuma muri porogaramu.

```Kin
ntahinduka x = 10
```

> Ntushobora guhindura agaciro k'ihinduragaciro `x` nyuma muri porogaramu.

> **Icyitonderwa:** Ugomba guha agaciro variable idahinduka, bityo ibi ntibizakora:

```Kin
ntahinduka x; # Bizatera ikibazo
```
