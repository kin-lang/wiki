# Intangiriro

**Variables** ni ibikoresho byo **kubika amakuru**.

Muri **_Kin_**, variables zishobora gutangazwa mu buryo bubiri:

## `reka`

Iyo utanze variable ukoresheje ijambo ry’ingenzi `reka`, ushobora guhindura agaciro ka variable nyuma muri porogaramu.

```Kin
reka x = value
```

Variable `x` iratangazwa kandi ihabwa agaciro `value`, ariko agaciro gashobora guhindurwa nyuma muri porogaramu.

Nanone, ushobora gutanga variable utayihaye agaciro:

```Kin
reka x;
```

> Muri iki gihe, variable `x` iratangazwa ariko nta gaciro yahawe.
> **Inyandiko:** Ugomba kongeraho semicolon **_(;)_** ku iherezo ry’iyi statement.

## `ntahinduka`

Iyo utanze variable ukoresheje ijambo ry’ingenzi `ntahinduka`, ntabwo ushobora guhindura agaciro ka variable nyuma muri porogaramu.

```Kin
ntahinduka x = 10
```

> Ntushobora guhindura agaciro ka variable `x` nyuma muri porogaramu.

> **Inyandiko:** Ugomba guha agaciro variable ihamye, bityo ibi ntibizakora:

```Kin
ntahinduka x; # Bizatera error
```
