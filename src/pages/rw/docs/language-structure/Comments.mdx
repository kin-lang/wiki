# Ibisobanuro

**_Kin_** comments zikoreshwa gusobanura code ya **_Kin_**, no gutuma isomeka neza.

**_Kin_** comments kandi zishobora gukoreshwa mu guhagarika ishyirwa mu bikorwa ry’iyo code igihe uri kugerageza indi code.

# Uko Bikora

Ibisobanuro bitangizwa na `#`.

Ibyo ari byo byose biri hagati ya `#` n’iherezo ry’umurongo bizirengangizwa na **_Kin_** (ntibizashyirwa mu bikorwa).

Urugero rukoreshwa comment mbere y’umurongo wa code:

```Kin
# Ibi ni ibisobanuro
reka x = 10
```

Urugero rukoreshwa comment ku murongo umwe ku iherezo ry’umurongo kugira ngo rusobanure code:

```Kin
reka x = 5      # Tangaza x, uyihere agaciro ka 5
```

> **Icyitonderwa:** **_Kin_** ntishyigikira comments z’imirongo myinshi (multi-line). Ugomba gukoresha `#` ku ntangiriro ya buri murongo.
