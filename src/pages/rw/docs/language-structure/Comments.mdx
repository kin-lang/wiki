# Ibisobanuro

**_Kin_** comments zikoreshwa gusobanura code ya **_Kin_**, no gutuma isomeka neza.

**_Kin_** comments kandi zishobora gukoreshwa mu guhagarika ishyirwa mu bikorwa ry’iyo code igihe uri kugerageza indi code.

# Uko Bikora

Comments zitangirana na `#`.

Icyo ari cyo cyose kiri hagati ya `#` n’iherezo ry’umurongo kizarekerwa na **_Kin_** (ntigizashyirwa mu bikorwa).

Urugero rukoreshwa comment mbere y’umurongo wa code:

```Kin
# Ibi ni ibisobanuro
reka x = 10
```

Urugero rukoreshwa comment ku murongo umwe ku iherezo ry’umurongo kugira ngo usobanure code:

```Kin
reka x = 5      # Tangaza x, uyihere agaciro ka 5
```

> **Inyandiko:** **_Kin_** ntishyigikira comments z’imizingo myinshi (multi-line). Ugomba gukoresha `#` ku ntangiriro y’umurongo wose.
