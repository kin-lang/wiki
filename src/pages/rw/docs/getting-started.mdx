## Kwinjiza (Installation)

Kin iboneka binyuze muri **npm package manager**, ushobora kuyinjiza ukoresheje:

```bash
# npm
npm i -g @kin-lang/kin

# yarn
yarn add -g @kin-lang/kin

# pnpm
pnpm add -g @kin-lang/kin
```

> Koresha `kin --version` urebe niba **_Kin_** yinjijwe neza.
> Ku bakoresha **_Linux Distros_** na **_macOS_**, ntuzibagirwe gukoresha **_sudo_**.

## Kwihuza na IDE (IDE Integrations)

Twatanze n'inyongera (extension) yemewe ku **Visual Studio Code** igufasha kugira uburambe bwiza mu kwandika muri Kin.

- [Injiza iva kuri VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=pacifiquem.kinlang)
- Cyangwa ushakishe ijambo `kinlang` mu isanduku yo gushakamo extensions muri VS Code, hanyuma winjize iy'itanga igisubizo cya mbere!

## Porogaramu ya Mbere

Kora dosiye ifite inyongera `.kin`, wandikemo aya mabwiriza kugira ngo porogaramu ya mbere **Hello World** ikore:

```kin
reka name = injiza_amakuru("Izina ryawe ni irihe? ")
tangaza_amakuru("Muraho, ", name, "!")
```

Koresha porogaramu yawe unyura aho dosiye iherereye kuri sisiteme yawe, hanyuma ukore:

```bash
kin run filename.kin
```

> Menya neza ko **Kin** yinjijwe kandi uri mu bubiko bukwiye (correct directory).
